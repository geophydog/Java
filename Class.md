## Java Class and sub-class inherit super-class
***
### :one: Class form
```java
public class ClassName {
  Attributes;
  ...
  Methods;
  ...
}
```
#### Example,
```java
public class Animal {
  String color;
  String name;
  public void eat() {
    System.out.println("Animal will eat something!");
  }
  
  public void breathe() {
    System.out.println("Animal can breathe!");
  }
}
```
